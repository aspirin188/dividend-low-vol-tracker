### 字段次序调整详情

#### 前端列表显示次序

**调整前 (v6.6):**
```
排名 | 代码 | 名称 | 行业 | 市场 | 股价 | PE | PB | 股息率(%) | 波动率(%) | 综合评分 | 总市值(亿) | 股利支付率(%) | EPS
```

**调整后 (v6.7):**
```
排名 | 代码 | 名称 | 行业 | 市场 | 股价 | PE | PB | 股息率(%) | 总市值(亿) | 综合评分 | 波动率(%) | 股利支付率(%) | EPS
```

#### Excel导出次序

**调整前 (v6.6):**
```
排名 | 代码 | 名称 | 行业 | 市场 | 股价 | PE | PB | 股息率(%) | 波动率(%) | 综合评分 | 总市值(亿) | 股利支付率(%) | EPS
```

**调整后 (v6.7):**
```
排名 | 代码 | 名称 | 行业 | 市场 | 股价 | PE | PB | 股息率(%) | 总市值(亿) | 综合评分 | 波动率(%) | 股利支付率(%) | EPS
```

#### 调整原因

1. **股息率、市值前置**: 这两个是核心筛选指标,应紧跟PB之后,便于用户快速判断
2. **综合评分前置**: 评分是核心排序依据,应放在波动率之前,突出重要性
3. **波动率后移**: 波动率是辅助参考指标,可适当后移

## 3. 两因子评分模型

```
综合评分 = 股息率归一化 × 0.6 + (1 - 波动率归一化) × 0.4

数据口径:
- 股息率:TTM(近12个月,自计算)
- 波动率:120日对数收益率,年化 √242

硬性筛选:
- 股息率 ≥ 4%
- 总市值 ≥ 100亿
- 股利支付率 ≤ 150%
- EPS > 0
- 非ST
```

## 5. 技术方案

### 5.1 数据表结构(16 字段)

```sql
CREATE TABLE stock_data (
    code             TEXT PRIMARY KEY,
    name             TEXT,
    industry         TEXT,       -- 归并后的大类行业
    market           TEXT,       -- 市场类型:沪市/深市/创业板/科创板
    dividend_yield   REAL,       -- 自计算TTM股息率
    annual_vol       REAL,
    composite_score  REAL,
    rank             INTEGER,
    market_cap       REAL,
    payout_ratio     REAL,
    eps              REAL,
    price            REAL,
    pe               REAL,
    pb               REAL,
    data_date        TEXT,
    updated_at       TEXT
);
```

**注意**: 数据库字段顺序不变,仅调整前端显示和Excel导出顺序

### 5.2 字段显示映射

