### 功能详情

#### 分红稳定性因子定义

**核心逻辑**：
- 只保留连续3年以上（含3年）有分红记录的股票
- 剔除只有单次或偶尔分红的公司
- 剔除"一次性大分红"凑股息率指标的投机行为

**数据来源**：
- 东方财富数据中心 RPT_LICO_FN_CPD 接口
- 查询每只股票近5年的分红方案记录
- 统计连续分红年份数

**筛选规则**：
```
硬性条件：连续分红年数 >= 3年
计算口径：过去5年内，有分红记录的年份数 >= 3
```

#### 三因子评分模型（v6.10升级）

```
综合评分 = 股息率归一化 × 0.5 + (1 - 波动率归一化) × 0.3 + 分红稳定性归一化 × 0.2

因子权重调整：
- 股息率：60% → 50%
- 波动率：40% → 30%
- 分红稳定性：新增 20%
```

**分红稳定性评分规则**：
| 连续分红年数 | 得分 |
|------------|------|
| 3年 | 60分 |
| 4年 | 80分 |
| 5年及以上 | 100分 |

#### 数据结构扩展

```sql
-- 数据库新增字段
ALTER TABLE stock_data ADD COLUMN dividend_years INTEGER;  -- 连续分红年数
```

#### 影响范围

## 3. 三因子评分模型（v6.10）

```
综合评分 = 股息率归一化 × 0.5 + (1 - 波动率归一化) × 0.3 + 分红稳定性归一化 × 0.2

数据口径:
- 股息率:TTM(近12个月,自计算)
- 波动率:120日对数收益率,年化 √242
- 分红稳定性:过去5年连续分红年数

硬性筛选:
- 股息率 ≥ 3%
- 总市值 ≥ 500亿
- 股利支付率 ≤ 150%
- EPS > 0
- 非ST
- 连续分红年数 ≥ 3年 ← v6.10新增
```
