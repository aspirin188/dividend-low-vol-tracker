### v6.7 → v6.8 修订记录

| 修订项 | 内容 | 原因 |
|--------|------|------|
| **股息率筛选下限调整** | 默认最低股息率从4%降至3% | 扩大筛选范围，覆盖更多优质标的 |
| **市值筛选下限调整** | 默认最低市值从100亿提升至500亿 | 聚焦大盘蓝筹，提升稳定性 |

### 筛选条件调整详情

#### 硬性筛选条件对比

| 筛选条件 | v6.7 | v6.8 | 说明 |
|----------|------|------|------|
| 股息率 | ≥ 4% | ≥ **3%** | 扩大覆盖范围 |
| 总市值 | ≥ 100亿 | ≥ **500亿** | 聚焦大盘蓝筹 |
| 股利支付率 | ≤ 150% | ≤ 150% | 保持不变 |
| EPS | > 0 | > 0 | 保持不变 |
| ST股票 | 排除 | 排除 | 保持不变 |

#### 调整原因

1. **股息率降至3%**:
   - 3%股息率仍显著高于银行理财收益率
   - 扩大筛选范围，覆盖更多优质蓝筹股
   - 部分优质公司股息率在3-4%区间，不应被排除

2. **市值提升至500亿**:
   - 500亿以上市值公司经营更稳定
   - 流动性更好，适合大资金配置
   - 降低小市值公司风险暴露

#### 影响范围

- **筛选代码**: `server/services/scorer.py` 中的 `MIN_DIVIDEND_YIELD` 和 `MIN_MARKET_CAP`
- **前端筛选器**: 默认选项需同步调整
- **数据量预期**: 股息率降低会增加候选股，市值提升会减少候选股，整体变化需实测

## 3. 两因子评分模型

```
综合评分 = 股息率归一化 × 0.6 + (1 - 波动率归一化) × 0.4

数据口径:
- 股息率:TTM(近12个月,自计算)
- 波动率:120日对数收益率,年化 √242

硬性筛选:
- 股息率 ≥ 3%  ← v6.8 调整
- 总市值 ≥ 500亿  ← v6.8 调整
- 股利支付率 ≤ 150%
- EPS > 0
- 非ST
```

## 5. 技术方案

### 5.1 数据表结构(16 字段)

```sql
CREATE TABLE stock_data (
    code             TEXT PRIMARY KEY,
    name             TEXT,
    industry         TEXT,       -- 归并后的大类行业
    market           TEXT,       -- 市场类型:沪市/深市/创业板/科创板
    dividend_yield   REAL,       -- 自计算TTM股息率
    annual_vol       REAL,
    composite_score  REAL,
    rank             INTEGER,
    market_cap       REAL,
    payout_ratio     REAL,
    eps              REAL,
    price            REAL,
    pe               REAL,
    pb               REAL,
    data_date        TEXT,
    updated_at       TEXT
);
```

### 5.2 筛选条件配置(v6.8)

```python
# server/services/scorer.py

MIN_DIVIDEND_YIELD = 3.0   # 股息率 ≥ 3% (v6.8调整: 4.0 → 3.0)
MIN_MARKET_CAP = 500.0      # 总市值 ≥ 500亿 (v6.8调整: 100.0 → 500.0)
MAX_PAYOUT_RATIO = 150.0    # 股利支付率 ≤ 150%
MIN_EPS = 0.0               # EPS > 0
```

### 5.3 API 筛选参数

```
/api/stocks 支持 query 参数:

  market_cap_range  市值区间
    "large"         ≥ 1000 亿
    "medium"        500 - 1000 亿  ← v6.8: 现为默认下限
    "small"         100 - 500 亿   ← v6.8: 现已低于默认值

  div_yield_min     股息率下限
    "3"             ← v6.8: 新增,现为默认下限
    "4" / "6" / "8"

  market            市场类型
    "sh"            沪市主板
    "sz"            深市主板
    "gem"           创业板
    "star"          科创板
```
