# V8.4.1 成长因子筛选增强方案

## 修订日期
2026-04-01

## 问题背景

V8.4 版本的成长因子仅作为排序加分项存在，不影响股票筛选结果。这导致：
- 负增长股票仍可能入选（只要股息高、波动低）
- 成长因子对选股结果影响有限
- 无法真正筛选出"持续成长的优质高股息公司"

## 改进目标

增加**最低利润增长筛选条件**，在筛选阶段过滤掉负增长公司，确保入选股票具备基本的盈利能力。

## 方案设计

### 1. 新增配置参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| MIN_PROFIT_GROWTH | float | 0 | 最低3年净利润CAGR（%），设为0表示不能为负 |
| ENABLE_PROFIT_GROWTH_FILTER | bool | True | 是否启用利润增长筛选 |

### 2. 筛选逻辑

```
候选股票池 → A类筛选 → [利润增长筛选] → B类风控 → 四因子评分
                                          ↑
                        过滤 profit_growth_3y < MIN_PROFIT_GROWTH
```

### 3. 筛选时机

在 B类风控 之前、A类筛选 之后执行。

### 4. 数据处理

- `profit_growth_3y` 为 None（无数据）的股票：
  - 默认放行（给中等分 30 分）
  - 或严格模式：视为不满足条件

### 5. 预设策略适配

| 策略 | MIN_PROFIT_GROWTH | 说明 |
|------|-------------------|------|
| 保守型 | 5 | 要求稳定增长 |
| 均衡型 | 3 | 要求正增长 |
| 激进型 | 0 | 不要求增长 |
| 高股息 | 0 | 不要求增长 |
| 低估型 | 3 | 要求正增长 |
| 成长红利 | 5 | 要求稳定增长 |
