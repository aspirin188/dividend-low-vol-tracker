## 一、改造背景与思路

### 1.1 现有问题分析

v6.20版本三因子模型（股息率50% + 波动率30% + 分红稳定性20%）存在以下核心问题：

| 问题 | 根因 |
|------|------|
| 股息率仅看TTM | 容易被一次性特别分红误导 |
| 波动率窗口偏短 | 120日约6个月，稳定性不足 |
| 支付率只看单年 | 容易受单年异常影响 |

### 1.2 简化思路

**第一性原理追问**：
- 质量因子的本质目的？→ 排雷，确保分红可持续
- 最简洁有效的排雷方式？→ 用**历史均值**平滑异常值

**极简主义改造**：
- **不新增**数据接口
- **不新增**计算函数
- **复用**现有数据，强化现有因子

## 二、改造方案：强化现有三因子

### 2.1 改造对比

| 因子 | v6.20 | v7.0 | 改进点 |
|------|-------|------|--------|
| 股息率 | TTM股息率 | **3年平均股息率** | 平滑异常分红 |
| 波动率 | 120日年化 | **252日年化** | 更稳定可靠 |
| 分红稳定性 | 单维度(年数) | **年数 + 3年平均支付率** | 验证分红真伪 |
| **新增数据接口** | - | **无** | 零成本 |

### 2.2 股息率改造：TTM → 3年平均

**原问题**：TTM股息率容易被一次性特别分红误导

**改造方案**：
```
DIV_YIELD_3Y = mean(DIV_2024, DIV_2023, DIV_2022)
```

**示例**：
```
某公司:
- 2022年: 分红 0.5元/股, 股价10元 → 股息率 5%
- 2023年: 分红 0.5元/股, 股价10元 → 股息率 5%
- 2024年: 分红 5.0元/股(含特别分红), 股价10元 → TTM股息率 50% ← 陷阱!

3年平均 = (5% + 5% + 50%) / 3 = 20%  ← 仍然偏高，但比TTM合理
```

**筛选逻辑**：
```python
# 3年平均股息率 >= 下限
# 且 TTM股息率不能超过平均值的3倍（排除极端异常）
if div_ttm > div_3y_avg * 3:
    标记为"异常高息" → 降权或过滤
```

### 2.3 波动率改造：120日 → 252日

**原问题**：120日约6个月，窗口偏短，容易受短期波动影响

**改造方案**：
```python
# 当前 (v6.20)
_VOL_WINDOW = 120  # 约6个月

# 改造后 (v7.0)
_VOL_WINDOW = 252  # 约1年
_VOL_ANNUALIZE = math.sqrt(242)
```

**理由**：
- A股牛熊周期约1年
- 252日约覆盖一个完整周期
- 与红利低波策略的"长期持有"理念一致

### 2.4 分红稳定性增强：加入3年平均支付率

**原问题**：单年支付率可能被操纵，无法验证"真金白银"

**改造方案**：
```
PAYOUT_3Y = mean(PAYOUT_2024, PAYOUT_2023, PAYOUT_2022)
```

**筛选逻辑**：
```
正常: PAYOUT_3Y 在 30%-100% 之间
警告: PAYOUT_3Y 在 100%-150% 之间 → 需要FCF验证
危险: PAYOUT_3Y > 150% → 高概率不可持续
```

**与现有支付率筛选的关系**：
```
现有: MAX_PAYOUT_RATIO = 150% (硬性过滤)
新增: PAYOUT_3Y 作为评分维度之一
     - 低且稳定(30%-80%) → 高分
     - 高但稳定(80%-100%) → 中分
     - 波动大或持续过高 → 低分
```

## 三、评分模型改造

### 3.1 权重分配

| 版本 | 股息率 | 波动率 | 分红稳定性 | 说明 |
|------|--------|--------|------------|------|
| v6.20 | 50% | 30% | 20% | 原三因子 |
| **v7.0** | **50%** | **25%** | **25%** | 稳定性权重提升 |

**改造理由**：
- 股息率保持50%，策略核心不变
- 波动率权重微降5%，因窗口延长后数据更稳定
- 分红稳定性权重提升10%，纳入支付率稳定性维度

### 3.2 分红稳定性内部评分

```
v6.20 稳定性评分:
- 3年 = 60分
- 4年 = 80分
- 5年+ = 100分

v7.0 稳定性评分:
稳定性总分 = 分红年数分 × 0.6 + 支付率稳定性分 × 0.4

分红年数分: (同v6.20)
- 3年 = 60分
- 4年 = 80分
- 5年+ = 100分

支付率稳定性分:
- PAYOUT_3Y 在 30%-80%: 100分
- PAYOUT_3Y 在 80%-100%: 80分
- PAYOUT_3Y 在 100%-150%: 50分
- PAYOUT_3Y > 150%: 0分
```

### 3.3 综合评分公式

```
composite_score = (
    div_3y_norm × 0.50 +           # 3年平均股息率
    vol_score × 0.25 +             # 波动率（低好）
    stability_score × 0.25         # 分红稳定性（增强版）
) × 100
```

## 七、风险与注意事项

### 7.1 风险评估

| 风险 | 概率 | 影响 | 缓解 |
|------|------|------|------|
| 历史分红数据不足3年 | 低 | 中 | 不足3年则用已有年份 |
| 252日窗口数据获取慢 | 低 | 低 | 候选股仅30只，可接受 |
| 特别分红跨年计算偏差 | 低 | 低 | TTM异常检测补偿 |

### 7.2 注意事项

1. **数据一致性**：股息率和支付率必须来自同一财年
2. **特别分红识别**：当TTM显著高于3年均值时，降权处理
3. **银行股适配**：银行股高支付率是正常的，保持豁免逻辑
