## 二、产品背景

### 2.1 产品定位

红利低波跟踪系统是一款基于价值投资理念的A股高股息股票筛选工具，旨在帮助投资者从全A股中筛选出兼具高股息、低波动、分红稳定特性的优质标的。系统采用多因子筛选模型，结合技术面信号分析，为投资者提供决策支持。

### 2.2 版本演进历程

| 版本 | 日期 | 核心改进 | 状态 |
|------|------|---------|------|
| v6.20 | 2026-03-29 | 参数化配置系统 | 已完成 |
| v7.0 | 2026-03-29 | 质量因子增强（支付率稳定性、波动率配置化） | 已完成 |
| v7.3 | 2026-03-30 | 信号系统升级（5级买入/4级卖出信号） | 已完成 |
| v7.5 | 2026-03-31 | 数据源完全重构（移除东方财富） | 已完成 |
| v7.6 | 2026-03-31 | 极简稳定版（性能优化、兼容性修复） | 已完成 |
| **v8.0** | **2026-03-31** | **数据增强、信号优化、配置升级** | **开发中** |

### 2.3 当前系统状态（v7.6）

#### 2.3.1 系统能力

v7.6版本已达到稳定运行状态，核心功能包括：
- 高股息筛选（股息率≥3%）
- 财务数据获取（EPS、ROE）
- 实时股价查询（新浪财经接口）
- 分红数据获取（akshare接口）
- 买卖信号计算（250日均线策略）
- 参数化配置系统

#### 2.3.2 性能指标

| 指标 | v7.6表现 | 说明 |
|------|---------|------|
| 数据获取耗时 | 14秒 | 较v7.3提升350倍 |
| 候选股数量 | 约247只 | 稳定输出 |
| 系统稳定性 | 高 | 完整测试通过 |
| 代码量 | 减少60% | 极简设计 |

#### 2.3.3 已知限制

基于v7.6版本，当前系统存在以下可改进点：
- 价格百分位字段暂时不可用（为None）
- 波动率字段暂时不可用（为None）
- 负债率字段暂时不可用（为None）
- 分红年数使用默认值5年
- 质量因子函数为兼容性桩函数，未实现真正功能

## 三、v8.0升级目标

### 3.1 升级背景

v7.6版本的极简设计虽然解决了稳定性和性能问题，但也做出了一些妥协，导致部分高级功能暂时不可用。v8.0版本将在保持v7.6稳定性的基础上，逐步恢复并增强这些功能，实现「稳定可靠」与「功能完整」的平衡。

### 3.2 核心目标

v8.0版本的核心目标包括四个方面：

**目标一：数据源增强**
- 恢复价格百分位计算功能
- 恢复波动率计算功能
- 恢复负债率数据获取
- 优化分红数据获取逻辑

**目标二：信号系统优化**
- 完善买卖信号计算逻辑
- 增加信号准确度验证
- 优化趋势判断算法

**目标三：配置系统升级**
- 增加预设策略模板
- 优化参数校验逻辑
- 增加参数历史记录

**目标四：质量因子回归**
- 实现真正的质量因子计算
- 恢复击球区评分功能
- 优化评分权重配置

### 3.3 升级路径

```
v7.6 (极简稳定版)
    ↓
v8.0 (数据增强版)
    ├─ 数据源增强 ✅
    ├─ 信号系统优化 ✅
```

#### 4.4.2 击球区评分恢复

**需求说明**：
击球区评分是综合估值、动量、趋势等因素的综合性评分，帮助投资者识别最佳买入时机。

**实现逻辑**：
```python
def calculate_strike_zone_score(stock: pd.Series) -> float:
    """
    计算击球区评分（0-100分）
    
    评分维度：
    1. 估值维度（PE、PB历史百分位）
    2. 动量维度（20日涨跌幅）
    3. 趋势维度（均线位置、趋势强度）
    4. 基本面维度（ROE、股息率）
    
    评分公式：
    strike_zone_score = (
        valuation_score * 0.25 +
        momentum_score * 0.25 +
        trend_score * 0.30 +
        fundamental_score * 0.20
    )
    """
```

**评级划分**：
| 评分区间 | 评级 | 颜色 | 说明 |
|---------|------|------|------|
| 80-100 | 极好 | 绿色 | 最佳击球区 |
| 60-80 | 较好 | 蓝色 | 可以考虑 |
| 40-60 | 一般 | 黄色 | 继续观察 |
| 20-40 | 较差 | 橙色 | 谨慎对待 |
| 0-20 | 极差 | 红色 | 回避 |

**优先级**：中
**预计工时**：2小时
