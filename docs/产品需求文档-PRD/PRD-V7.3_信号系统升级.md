## 一、版本概览

### 1.1 版本定位

v7.3是一次**信号系统完善升级**，基于250日均线策略，实现完整的买卖信号提示体系，帮助用户识别买入时机、卖出时机和持有状态。

### 1.2 核心改进

| 改进项 | v7.2.1 | v7.3 | 价值 |
|--------|--------|------|------|
| 趋势判断 | 斜率正负 | **30日对比判断** | 更准确的趋势识别 |
| 买入信号 | 4级分级 | **5级分级** | 更精细的买入时机 |
| 卖出信号 | 无 | **4级分级** | 风险控制闭环 |
| 信号展示 | 单列 | **3列（信号+趋势+均线位置）** | 更直观的决策支持 |
| 死叉检测 | 无 | **MA20/MA60死叉** | 强制卖出信号 |

### 1.3 升级路径

```
v7.2.1 (均线位置基础版)
```

## 二、策略框架

### 2.1 核心逻辑

**选股层（价值投资）**：
```
高股息 + 低波动 + 稳定分红 + 利润真实 → 找到好公司
```

**趋势层（250日线过滤）**：
```
250日线向上 + 价格在线上 → 有上升趋势 → 买入候选
250日线向下 + 价格在线下 → 下降趋势 → 排除/卖出
```

**买点层（回踩确认）**：
```
价格回踩250日线企稳 → 买入时机
```

**卖出层（趋势破坏）**：
```
价格有效跌破250日线 → 触发轮出
均线开始向下倾斜 → 确认趋势破坏
```

### 2.2 趋势判断算法

**方向判断（简化版）**：
```python
MA250_今日 > MA250_30日前 * 1.01 → 向上  # 30日上涨1%以上
MA250_今日 < MA250_30日前 * 0.99 → 向下  # 30日下跌1%以上
其他 → 横盘
```

**强度分级**：
```
强势：价格在线上15%以上 + 均线向上
温和：价格在线上0-15% + 均线向上
弱势：价格在线下0-10% + 均线向上（反弹中）
下降：价格在线下 + 均线向下
```

## 三、信号体系

### 3.1 买入信号（5级分级）

| 信号 | 条件 | 强度 | 操作建议 |
|------|------|------|---------|
| **强烈买入** ⭐⭐⭐⭐⭐ | 回踩均线±3% + 均线向上 + 强势趋势 | 5 | 黄金买点 |
| **买入** ⭐⭐⭐⭐ | 价格在均线上方0-5% + 均线向上 | 4 | 可买入 |
| **试探买入** ⭐⭐⭐ | 价格在均线上方5-10% + 温和趋势 | 3 | 小仓位试探 |
| **观望** ⭐⭐ | 价格在均线下方但接近 + 均线向上 | 2 | 等待回踩确认 |
| **不建议** ⭐ | 价格在均线下方 + 均线向下 | 0 | 不建议介入 |

### 3.2 卖出信号（4级分级）

| 信号 | 条件 | 紧急度 | 操作建议 |
|------|------|--------|---------|
| **强制卖出** 📉📉📉📉 | MA20 < MA60（短期死叉） | 高 | 不计成本卖出 |
| **清仓** 📉📉📉 | 价格跌破均线 + 均线向下 | 高 | 全部卖出 |
| **减仓** 📉📉 | 价格跌破均线 + 均线横盘 | 中 | 减半仓 |
| **警示** 📉 | 价格跌破均线 + 均线仍向上 | 低 | 密切观察 |

### 3.3 持有信号

| 信号 | 条件 | 操作建议 |
|------|------|---------|
| **持有** 📊 | 价格在均线上方 + 趋势向上 + 价格百分位>5% | 继续持有 |

### 3.4 观望信号

| 信号 | 条件 | 操作建议 |
|------|------|---------|
| **观望** ⏸️ | 等待明确信号 | 观望 |

## 五、技术实现

### 5.1 核心函数：`calc_ma_position_batch()`

**位置**：`server/services/fetcher.py`

**改进点**：
1. **趋势判断**：对比30日前MA250，判断趋势方向
2. **趋势强度**：结合价格位置和均线方向，判断趋势强度
3. **买入信号**：5级分级（强烈买入/买入/试探买入/观望/不建议）
4. **卖出信号**：4级分级（强制卖出/清仓/减仓/警示）
5. **新增MA20/MA60**：用于判断短期死叉

**代码示例**：
```python
# 趋势方向判断
ma_change_30d = (ma250 - ma250_30d_ago) / ma250_30d_ago

if ma_change_30d > 0.01:  # 30日上涨1%以上
    trend = '向上'
elif ma_change_30d < -0.01:  # 30日下跌1%以上
    trend = '向下'
else:
    trend = '横盘'

# 卖出信号优先级最高（风险控制第一）
if ma20 < ma60:
    signal = "强制卖出"
    signal_level = -4
    signal_type = "sell"
    action = "短期均线死叉，不计成本卖出"
```

### 5.2 前端展示优化

## 六、使用说明

### 6.1 如何解读信号

**买入信号解读**：
```
强烈买入：最佳买点，回踩250日线企稳
买入：价格在均线上方且趋势向上
试探买入：趋势温和，可小仓位试探
观望：等待更好的时机
不建议：趋势向下，不建议介入
```

**卖出信号解读**：
```
强制卖出：短期均线死叉，必须卖出
清仓：跌破250日线且趋势向下，全部卖出
减仓：跌破均线但趋势不明，减半仓
警示：跌破均线但趋势仍向上，观察
```

**持有信号解读**：
```
持有：价格在均线上方，趋势向上，继续持有
```

### 6.2 决策建议

**买入决策**：
1. 优先关注"强烈买入"信号
2. "买入"信号可作为标准买点
3. "试探买入"需要结合其他因素

**卖出决策**：
1. "强制卖出"必须执行
2. "清仓"信号需果断行动
3. "减仓"信号需密切观察
4. "警示"信号可暂时观察

**持有决策**：
1. "持有"信号继续持有
2. 定期检查趋势是否改变

## 八、风险提示

### 8.1 信号局限性

**不保证准确性**：
- 信号仅基于技术指标，不保证准确性
- 市场存在不确定性，信号可能失效

**需要结合基本面**：
- 信号仅是技术面参考
- 需要结合公司基本面、行业前景等因素

### 8.2 使用建议

**不要盲目跟随**：
- 信号是辅助决策工具，不是投资建议
- 需要结合自身风险偏好和投资策略

**严格执行纪律**：
- 一旦决策，严格执行
- 不要因为情绪而违背系统信号
