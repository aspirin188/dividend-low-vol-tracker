### Bug详细说明

#### 问题发现

用户反馈招商银行股息率异常:
- **系统显示**: 6.62%
- **正确计算**: 5.11%
- **差异**: +1.51%

#### 根因分析

1. **数据源问题**: 系统直接使用东方财富push2接口的f115字段(股息率TTM)
2. **f115字段不可靠**: 该字段计算逻辑不透明,数据准确性无法保证
3. **招商银行案例**:
   - TTM分红总额: 每股2.016元(2025年中期1.013元 + 年报1.003元)
   - 当前股价: 39.44元
   - 正确股息率: 2.016 ÷ 39.44 × 100 = 5.11%
   - f115返回: 6.62%(错误)

**废弃f115字段,改为自计算**:

## 3. 两因子评分模型

```
综合评分 = 股息率归一化 × 0.6 + (1 - 波动率归一化) × 0.4

数据口径:
- 股息率:TTM(近12个月,自计算) ← v6.6 修复
- 波动率:120日对数收益率,年化 √242

硬性筛选:
- 股息率 ≥ 4%
- 总市值 ≥ 100亿
- 股利支付率 ≤ 150%
- EPS > 0
- 非ST
```
