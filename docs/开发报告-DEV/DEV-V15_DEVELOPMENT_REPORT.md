**改进措施**:
- 增加了银行股权益字段识别范围
- 实现了同时获取利润表和资产负债表的逻辑
- ROE = 净利润 / 股东权益 * 100%

**建议**:
- 使用akshare yjbb作为ROE主要数据源（已实现，99.4%完整性）
