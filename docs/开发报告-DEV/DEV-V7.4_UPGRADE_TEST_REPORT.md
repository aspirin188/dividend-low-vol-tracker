## 三、升级内容详解

### 3.1 主要改动

**废弃的代码**:
```python
# fetch_all_quotes() - 东方财富 push2 接口
_EM_PUSH2_URL = 'https://push2.eastmoney.com/api/qt/clist/get'
```

**新增的代码**:
```python
# 新浪财经接口
_SINA_BASE_URL = 'http://hq.sinajs.cn/list='

# 新增函数
def _fetch_stock_list() -> list:
    """获取全A股股票列表（使用akshare）"""

def _fetch_quotes_batch_sina(stock_codes: list, batch_size: int = 100) -> list:
    """分批从新浪财经获取实时行情"""
```

### 3.2 数据获取流程

**旧流程（东方财富）**:
```
全量请求 → 分页获取 → 解析JSON → 返回DataFrame
问题：连接失败率高
```

**新流程（新浪）**:
```
获取股票列表 → 分批请求 → 解析文本 → 合并数据 → 返回DataFrame
优势：稳定性高、速度快
```

### 3.3 数据字段差异

**东方财富 push2 接口提供的字段**:
- ✅ 股价 (f2)
- ✅ PE (f9)
- ✅ PB (f23)
- ✅ 总市值 (f20)
- ✅ 行业 (f100)

**新浪财经接口提供的字段**:
- ✅ 股价
- ✅ 名称
- ✅ 成交量
- ✅ 成交额
- ❌ PE/PB（需要从其他数据源补充）
- ❌ 市值（需要从其他数据源补充）
- ❌ 行业（需要从其他数据源补充）

**补充方案**:
- PE/PB/市值/行业 → 通过 akshare 接口补充（已有）
