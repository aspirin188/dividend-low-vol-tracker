## 📖 文档命名规范

所有文档采用统一命名格式:

```
文档功能-V版本号.描述.md
```

### 命名示例

- **PRD-V8.0_红利低波跟踪系统.md** - 产品需求文档,V8.0版本
- **DEV-V7.6_FINAL_RELEASE.md** - 开发报告,V7.6最终发布
- **VERSION-V7.6_FINAL_SUMMARY.md** - 版本总结,V7.6最终总结
- **PERF-PERFORMANCE_DIAGNOSIS_REPORT.md** - 性能优化,性能诊断报告
- **TECH-DATASOURCE_COMPLETE_GUIDE.md** - 技术专题,数据源完整指南

## 📝 文档维护

### 添加新文档

1. 根据文档类型选择合适的目录
2. 按命名规范命名文件
3. 更新本文档的文档索引
4. 在对应的版本总结中添加引用

### 文档归档

- 过时版本文档保留在对应目录中,便于历史追溯
- 重大版本变更创建新的版本总结
- 定期清理重复和临时文件
