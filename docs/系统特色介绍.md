# 红利低波跟踪系统 - 系统特色介绍

> 一款为价值投资者打造的智能选股工具，通过三因子模型筛选高质量高股息股票

---

## 🎯 一、系统定位

### 1.1 核心价值

**帮助投资者寻找"三好股票"：好价格、好公司、好分红**

- **好价格**：高股息率代表股价相对便宜
- **好公司**：低波动率代表经营稳健
- **好分红**：连续分红代表回报股东意愿

### 1.2 适用人群

✅ 适合：
- 追求稳健收益的长期投资者
- 注重分红回报的价值投资者
- 希望降低投资风险的保守型投资者

❌ 不适合：
- 追求短期暴利的投机者
- 热衷概念炒作的激进投资者
- 频繁交易的短线操作者

## 🔍 二、选股理念

### 2.1 投资哲学

**"红利低波"策略**源于学术界和业界的长期研究：

1. **红利效应**：高股息股票长期跑赢大盘
2. **低波效应**：低波动率股票风险调整后收益更高
3. **质量因子**：盈利稳定、分红连续的公司质量更高

### 2.2 三因子模型

```
综合评分 = 股息率因子 × 50% + 波动率因子 × 30% + 稳定性因子 × 20%
```

**为什么是这个权重？**

- **股息率50%**：核心因子，直接反映投资回报
- **波动率30%**：风险因子，越低越稳健
- **稳定性20%**：质量因子，验证分红可持续性

## 📊 三、选股因子详解

### 3.1 股息率因子（权重50%）

#### 定义
**TTM股息率** = 近12个月每股分红总和 ÷ 当前股价 × 100%

#### 计算逻辑
```
1. 获取近12个月的分红记录
2. 累加所有每股分红金额
3. 除以当前股价
4. 转换为百分比
```

#### 评分方法
- **Min-Max归一化**：将所有股票的股息率映射到0-100分
- 股息率越高，得分越高
- 相对排名，而非绝对值

#### 为什么选TTM股息率？
- ✅ 最新的分红数据，反映当前情况
- ✅ 滚动12个月，平滑季节性波动
- ✅ 市场公认标准，便于对比

**案例**：
```
股票A: 股息率5%
股票B: 股息率3%
股票C: 股息率4%

归一化后：
股票A: 100分（最高）
股票C: 50分（中间）
股票B: 0分（最低）
```

---

### 3.2 波动率因子（权重30%）

#### 定义
**年化波动率** = 过去120日对数收益率的标准差 × √242

#### 计算逻辑
```
1. 获取过去120个交易日的收盘价
2. 计算每日对数收益率 = ln(今日收盘/昨日收盘)
3. 计算标准差
4. 年化处理：标准差 × √242
```

#### 评分方法
- **反向评分**：波动率越低，得分越高
- **Min-Max归一化**：映射到0-100分
- 取反逻辑：`波动率得分 = 1 - 归一化值`

#### v7.0新增特性
**波动率窗口可配置**：
- **120日**（默认）：约6个月，响应快，适合波段操作
- **252日**：约1年，更稳定，适合长期持有

#### 为什么选低波动率？
- ✅ 低波动股票风险调整后收益更高
- ✅ 心理压力小，拿得住
- ✅ 跌幅小，回撤可控

**案例**：
```
股票A: 年化波动率15%
股票B: 年化波动率25%
股票C: 年化波动率20%

归一化后（取反）：
股票A: 100分（最低波动，最高分）
股票C: 50分
股票B: 0分（最高波动，最低分）
```

---

### 3.3 稳定性因子（权重20%）⭐ v7.0核心升级

#### 定义
**分红稳定性** = 分红连续性 + 支付率可持续性

#### v7.0评分升级

**旧版（v6.20）**：
```
稳定性评分 = 分红年数评分
- 5年及以上: 100分
- 4年: 80分
- 3年: 60分
- 3年以下: 0分
```

**新版（v7.0）**：
```
稳定性评分 = 分红年数评分 × 60% + 支付率稳定性评分 × 40%
```

#### 分红年数评分（60%权重）

**连续分红年数**：过去5年每年都有现金分红

```
连续5年分红 → 100分
连续4年分红 → 80分
连续3年分红 → 60分
少于3年 → 0分
```

**意义**：
- 验证公司回报股东的意愿
- 体现盈利的稳定性
- 分红政策的一致性

#### 支付率稳定性评分（40%权重）⭐ v7.0新增

**支付率** = 每股分红 ÷ 每股收益 × 100%

**3年平均支付率评分**：
```
30% - 80%  → 100分（理想区间）
80% - 100% → 80分（合理区间）
100% - 150% → 50分（警告区间）
>150%      → 0分（危险区间）
```

**意义**：
- ✅ 验证分红是否"真金白银"
- ✅ 识别支付率超100%的不可持续分红
- ✅ 平滑单年异常值

**典型案例**：
```
某公司分红历史：
- 2022年: 支付率60%（正常）
- 2023年: 支付率70%（正常）
- 2024年: 支付率150%（异常！）

v6.20: 单年150% → 通过筛选（<150%上限）❌
v7.0: 3年平均93% → 中等评分，发出警告 ✅
```

## 🛡️ 四、硬性筛选条件

### 4.1 筛选逻辑

**"宁可错过，不可做错"**

系统在评分之前，先进行硬性条件筛选，过滤掉不符合标准的股票。

### 4.2 筛选标准

| 条件 | 标准 | 说明 |
|------|------|------|
| **股息率** | ≥ 3% | 确保有足够回报 |
| **股息率上限** | ≤ 10% | 避免异常高股息陷阱 |
| **总市值** | ≥ 500亿 | 确保流动性充足 |
| **股利支付率** | ≤ 150% | 避免不可持续分红 |
| **每股收益** | > 0 | 排除亏损公司 |
| **ST标识** | 无 | 排除风险警示股 |
| **连续分红** | ≥ 3年 | 验证分红意愿 |
| **ROE** | ≥ 8% | 筛选盈利能力 |
| **资产负债率** | ≤ 70% | 控制财务风险 |
| **资产负债率（金融地产）** | ≤ 85% | 行业特殊处理 |

### 4.3 筛选原则

1. **数据不全者过滤**：宁可少选，不可错选
2. **风险优先**：先排除风险，再追求收益
3. **行业差异**：金融地产负债率单独处理

## 🎨 七、v7.0核心升级亮点

### 7.1 支付率稳定性维度 ⭐⭐⭐⭐⭐

**问题**：
- 单年支付率易被操纵
- 支付率>100%不可持续
- 无法验证分红质量

**解决方案**：
- 计算3年平均支付率
- 分4档评分（30%-80% / 80%-100% / 100%-150% / >150%）
- 平滑异常值，识别真实分红能力

**价值**：
- ✅ 更准确评估分红可持续性
- ✅ 识别一次性特别分红
- ✅ 提升选股质量

### 7.2 波动率窗口配置化 ⭐⭐⭐⭐

**问题**：
- 固定120日可能不适合所有投资风格
- 短期投资者需要更快响应
- 长期投资者需要更稳定指标

**解决方案**：
- 120日（默认）：适合波段操作
- 252日：适合长期持有
- 用户可在配置页面自由切换

**价值**：
- ✅ 灵活适应不同投资风格
- ✅ 提升系统适用性

### 7.3 股息率异常检测 ⭐⭐⭐

**问题**：
- TTM股息率容易被一次性特别分红误导
- 例如：某公司股息率5%+5%+50%
- 平均后20%，但仍偏高

**解决方案**：
- 检测TTM股息率/3年均值比值
- 比值>3.0 → 特别分红（降权）
- 比值2.0-3.0 → 高波动（关注）
- 比值<2.0 → 正常

**价值**：
- ✅ 避免踩雷一次性分红
- ✅ 提升选股安全性

## 📚 十一、术语解释

### 11.1 核心术语

| 术语 | 英文 | 解释 |
|------|------|------|
| **股息率** | Dividend Yield | 每股分红/股价×100% |
| **TTM** | Trailing Twelve Months | 滚动12个月 |
| **波动率** | Volatility | 收益率标准差，衡量风险 |
| **支付率** | Payout Ratio | 每股分红/每股收益×100% |
| **ROE** | Return on Equity | 净资产收益率 |
| **EPS** | Earnings Per Share | 每股收益 |
| **PE** | Price to Earnings | 市盈率 |
| **PB** | Price to Book | 市净率 |

### 11.2 评分术语

| 术语 | 解释 |
|------|------|
| **归一化** | 将不同量纲的数值映射到0-1区间 |
| **Min-Max归一化** | (值-最小值)/(最大值-最小值) |
| **综合评分** | 三因子加权平均后的总分 |
| **股价百分位** | 当前股价在历史中的相对位置 |
