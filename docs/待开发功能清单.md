## 🟡 四、策略增强（优先级：中）

### 4.1 自由现金流收益率（v7.1规划）

**问题描述**：
- 分红质量验证维度不足
- 支付率无法完全验证分红可持续性
- 需要更全面的财务健康度评估

**待办事项**：
- [ ] 获取自由现金流数据
- [ ] 计算自由现金流收益率 = FCF / 市值
- [ ] 计算自由现金流支付率 = 分红 / FCF
- [ ] 集成到评分模型
- [ ] 更新PRD到v7.1

**核心价值**：
```
自由现金流验证:
├─ 验证分红资金来源
│  ├─ FCF支付率 < 100% → 分红有保障
│  └─ FCF支付率 > 100% → 分红不可持续
│
├─ 验证盈利质量
│  ├─ FCF > 净利润 → 盈利质量高
│  └─ FCF < 净利润 → 盈利质量差
│
└─ 新增评分维度
   └─ 稳定性因子可加入FCF稳定性
```

**预计工时**：8小时

**难度**：⭐⭐⭐⭐
