## 系统输出字段说明

### 一、基本面指标（核心评分维度）

#### 1. 股息率（dividend_yield）
- **数据口径**: TTM（最近12个月）自计算股息率
- **权重**: 35%（四因子模型中权重最高）
- **理想范围**: 3%-10%
- **含义**: 年度分红总额 / 当前股价，反映股东回报率
- **数值解读**:
  - 3%-5%: 合理水平，稳健分红
  - 5%-7%: 较高水平，优质标的
  - 7%-10%: 高分红，需警惕分红陷阱
  - >10%: 可能是特别分红，需深入分析

**示例**:
- 股息率 5.2% → 每持有1万元，年度分红约520元
- 股息率 3.8% → 每持有1万元，年度分红约380元

#### 2. 年化波动率（annual_vol）
- **数据口径**: 252日对数收益率年化波动率
- **权重**: 25%（反因子，越低越好）
- **理想范围**: <20%
- **含义**: 股价波动程度，反映风险水平
- **数值解读**:
  - <15%: 极低波动，防御性股票（银行、公用事业）
  - 15%-20%: 低波动，稳健股票（大金融、龙头消费）
  - 20%-30%: 中等波动，平衡股票（周期性龙头）
  - >30%: 高波动，需谨慎

**示例**:
- 波动率 12.5% → 价格稳定，适合保守型投资者
- 波动率 25.8% → 波动较大，需控制仓位

#### 3. 综合评分（composite_score）
- **计算公式**:
  ```
  综合评分 = 股息率因子×35% + 波动率因子×25% + 稳定性因子×15% + 成长因子×25%
  ```
- **评分范围**: 0-100分（归一化处理）
- **含义**: 多因子量化模型的综合质量评分
- **数值解读**:
  - 80-100分: 顶级标的，强烈推荐
  - 70-79分: 优质标的，建议配置
  - 60-69分: 合格标的，可以关注
  - <60分: 质量一般，谨慎选择

#### 4. 分红年数（dividend_years）
- **含义**: 连续分红年数
- **稳定性评分**:
  - ≥5年 → 100分（优秀）
  - 4年 → 80分（良好）
  - 3年 → 60分（及格）
  - <3年 → 0分（不合格）

**示例**:
- 分红年数 8年 → 长期稳定分红，值得信赖
- 分红年数 3年 → 刚满足最低要求，需关注持续性

#### 5. ROE（净资产收益率）
- **数据口径**: 加权净资产收益率
- **理想范围**: ≥10%（筛选条件）
- **含义**: 股东投入每100元能产生的净利润
- **数值解读**:
  - >20%: 极高盈利能力（优质银行、消费龙头）
  - 15%-20%: 高盈利能力（优秀公司）
  - 10%-15%: 良好盈利能力
  - <10%: 盈利能力较弱

**示例**:
- ROE 18.5% → 股东投入100元，年产生18.5元净利润
- ROE 8.2% → 低于筛选标准，被排除

#### 6. 市值（market_cap）
- **筛选条件**: ≥500亿（保证流动性）
- **含义**: 总市值，反映公司规模
- **数值解读**:
  - >10000亿: 超大蓝筹（银行、能源巨头）
  - 1000-10000亿: 大盘蓝筹
  - 500-1000亿: 中大盘股

#### 7. 股利支付率（payout_ratio）
- **含义**: 分红总额 / 净利润
- **筛选条件**: ≤150%
- **理想范围**: 30%-80%
- **数值解读**:
  - 30%-50%: 保守分红，留存再投资（成长性高）
  - 50%-80%: 合理分红，平衡发展
  - 80%-100%: 高分红，增长有限
  - 100%-150%: 警告区间，可能不可持续
  - >150%: 危险区间，分红来自历史利润

## 二、技术信号指标

### 1. 信号级别（signal_level）⭐ 核心指标

**10级买卖信号体系**:

#### 买入信号（5级）

| 级别 | 信号名称 | 信号级别 | 条件 | 强度 | 操作建议 |

### 2. 趋势（trend）

| 趋势 | 条件 | 含义 | 操作建议 |
|------|------|------|----------|
| **向上** | MA250今日 > MA250_30日前 × 1.01 | 上升趋势 | 重点关注买入信号 |
| **向下** | MA250今日 < MA250_30日前 × 0.99 | 下降趋势 | 避免买入，考虑卖出 |
| **横盘** | 其他 | 震荡期 | 谨慎操作，等待突破 |

### 3. 价格相对MA250（price_vs_ma_pct）

- **含义**: (当前价格 - MA250) / MA250 × 100%
- **数值解读**:
  - >15%: 强势区，距离买入时机较远
  - 5%-15%: 稳健区，可考虑建仓
  - -3%至5%: 回踩区，黄金买点⭐
  - -10%至-3%: 轻微跌破，关注反弹
  - <-10%: 有效跌破，趋势破坏

**示例**:
- 价格相对MA250 +2.3% → 在MA250上方，趋势良好
- 价格相对MA250 -1.8% → 回踩MA250附近，可能企稳
- 价格相对MA250 -12.5% → 有效跌破，风险较大

### 4. MA250（250日均线）

- **含义**: 过去250个交易日平均价格
- **重要性**: 牛熊分界线，长期趋势的核心指标
- **使用方法**:
  - 价格在MA250上方 + 均线向上 → 上升趋势
  - 价格在MA250下方 + 均线向下 → 下降趋势
  - 价格回踩MA250 → 潜在买入点

### 5. MA20和MA60

- **MA20**: 短期均线（20日）
- **MA60**: 中期均线（60日）
- **死叉信号**: MA20 < MA60 → 短期走弱，可能反转
- **金叉信号**: MA20 > MA60 → 短期走强

**示例**:
- MA20: 33.2, MA60: 33.0 → 金叉，短期走强 ✅
- MA20: 32.5, MA60: 33.2 → 死叉，短期走弱 ❌（signal_level = -4）

## 操作建议体系

### 一、买入操作指南

#### 1. 强烈买入（signal_level = 5）⭐⭐⭐⭐⭐

**触发条件**:
- 价格回踩MA250±3%（±3%范围内）
- MA250趋势向上
- 价格百分位>5%（强势趋势）

**操作建议**:
```
仓位建议: 重仓买入（占总资金40%-60%）
建仓方式: 一次性建仓或分2次建仓（70%+30%）
持有周期: 中长期（6-12个月以上）
```

**实战案例**:
```
股票: 某银行股
MA250: 5.20元
当前价格: 5.15元
价格相对MA250: -0.96% (在±3%回踩区间)
趋势: 向上
信号级别: 5 (强烈买入)

操作: 重仓买入，目标持有1年
```

#### 2. 买入（signal_level = 4）⭐⭐⭐⭐

**触发条件**:
- 价格在MA250上方0-5%
- MA250趋势向上

**操作建议**:
```
仓位建议: 建仓买入（占总资金20%-40%）
建仓方式: 分2-3次建仓（40%+30%+30%）
持有周期: 中期（3-6个月）
```

**实战案例**:
```
股票: 某保险股
MA250: 28.50元
当前价格: 29.20元
价格相对MA250: +2.46% (在0-5%区间)
趋势: 向上
信号级别: 4 (买入)

操作: 建仓30%，等待回调加仓
```

#### 3. 试探买入（signal_level = 3）⭐⭐⭐

**触发条件**:
- 价格在MA250上方5-10%
- 温和趋势

**操作建议**:
```
仓位建议: 小仓位试探（占总资金10%-20%）
建仓方式: 一次性小仓
持有周期: 短期观察（1-3个月）
```

**实战案例**:
```
股票: 某公用事业股
MA250: 12.80元
当前价格: 13.65元
价格相对MA250: +6.64% (在5-10%区间)
趋势: 向上（温和）
信号级别: 3 (试探买入)

操作: 小仓位试探10%，等待回踩再加仓
```

#### 4. 观望（signal_level = 2）⭐⭐

**触发条件**:
- 价格在MA250下方但接近
- MA250趋势向上

**操作建议**:
```
仓位建议: 暂不建仓
操作策略: 等待价格回踩MA250±3%区域
观察周期: 1-2周
```

**实战案例**:
```
股票: 某能源股
MA250: 15.30元
当前价格: 14.85元
价格相对MA250: -2.94% (接近MA250)
趋势: 向上
信号级别: 2 (观望)

操作: 暂不买入，等待价格回踩MA250区域（14.84-15.77元）
```

### 二、卖出操作指南

#### 1. 强制卖出（signal_level = -4）📉📉📉📉

**触发条件**: MA20 < MA60（短期死叉）

**含义**: 短期趋势反转，可能有较大下跌风险

**操作建议**:
```
紧急程度: 极高
卖出方式: 不计成本卖出，不要犹豫
减仓比例: 全部清仓
```

**实战案例**:
```
股票: 某券商股
MA20: 18.20元
MA60: 18.50元
MA20 < MA60: 是（死叉）
信号级别: -4 (强制卖出)

操作: 立即全部卖出，不要等待反弹
```

#### 2. 清仓（signal_level = -3）📉📉📉

**触发条件**:
- 价格跌破MA250
- MA250趋势向下

**含义**: 长期趋势被破坏，进入下降通道

**操作建议**:
```
紧急程度: 高
卖出方式: 分批卖出，或一次性清仓
减仓比例: 全部清仓
```

**实战案例**:
```
股票: 某周期股
MA250: 25.60元
当前价格: 23.80元
价格相对MA250: -7.03% (跌破MA250)
趋势: 向下
信号级别: -3 (清仓)

操作: 全部清仓，退出观望
```

#### 3. 减仓（signal_level = -2）📉📉

**触发条件**:
- 价格跌破MA250
- MA250横盘（趋势不明）

**含义**: 趋势不明，风险上升，建议降低仓位

**操作建议**:
```
紧急程度: 中
卖出方式: 减半仓
减仓比例: 50%
```

**实战案例**:
```
股票: 某消费股
MA250: 38.40元
当前价格: 36.20元
价格相对MA250: -5.73% (跌破MA250)

```

#### 4. 警示（signal_level = -1）📉

**触发条件**:
- 价格跌破MA250
- MA250仍向上（可能只是短期回调）

**含义**: 短期跌破，需密切观察

**操作建议**:
```
紧急程度: 低
操作策略: 密切观察，不急于卖出
观察重点: 是否有效跌破MA250（持续3个交易日以上）
```

**实战案例**:
```

MA250: 52.30元
当前价格: 50.80元
价格相对MA250: -2.87% (跌破MA250)
趋势: 向上
信号级别: -1 (警示)

操作: 密切观察，等待3个交易日确认是否有效跌破
```

## 注意事项

### 1. 信号优先级

**卖出信号优先级最高**（风险控制第一）:
```
1. 强制卖出（-4） → 立即执行
2. 清仓（-3） → 立即执行
3. 减仓（-2） → 尽快执行
4. 警示（-1） → 密切观察
```

**买入信号需要结合基本面**:
```
1. 强烈买入（5） + 综合评分>70 → 重仓
2. 强烈买入（5） + 综合评分60-70 → 中等仓位
3. 买入（4） + 综合评分>75 → 建仓
4. 买入（4） + 综合评分<70 → 观望
```

### 2. 不要单独依赖信号

**信号系统的局限性**:
- 只反映价格趋势，不反映公司基本面
- 可能滞后（技术指标都存在滞后性）
- 可能出现假信号（假突破、假跌破）

**正确使用方法**:
- 信号 = **择时工具**（解决何时买卖）
- 基本面 = **选股工具**（解决买什么）
- 两者结合才能做出正确决策

**决策流程**:
```
第一步: 基本面筛选（综合评分>60）✅
第二步: 技术信号判断（signal_level >= 3）✅
第三步: 风险控制（年化波动率<25%）✅
第四步: 仓位管理（根据信号级别调整仓位）✅
```

### 3. 特殊情况处理

**情况1: 高股息但信号差**
```
问题: 股息率8%，但signal_level = -1（警示）
分析: 可能是价值陷阱
操作: 不买入，等待信号改善
```

**情况2: 信号好但波动率高**
```
问题: signal_level = 5（强烈买入），但年化波动率35%
分析: 技术面好但风险高
操作: 降低仓位（不要超过总资金的20%）
```

**情况3: 多只股票同时满足买入条件**
```
问题: 5只股票signal_level都>=4
操作:
- 优先选择综合评分最高的2-3只
- 控制总体仓位（不要超过总资金的60%）
- 分散投资（不同行业、不同主题）
```

### 4. 风险控制原则

**三大铁律**:
1. **亏损>15%立即止损**（不要抱有幻想）
2. **单一股票不超过总资金的30%**（分散风险）
3. **总仓位不超过80%**（保留现金储备）

**仓位管理**:
```
总资金: 100万元
- 强烈买入（5）股票1: 30万元
- 买入（4）股票2: 20万元
- 买入（4）股票3: 20万元
- 现金储备: 30万元
```

## 总结

### 信号级别速查表

```
买入信号（从强到弱）:
5 - 强烈买入 → 重仓（40%-60%）
4 - 买入 → 建仓（20%-40%）
3 - 试探买入 → 小仓（10%-20%）
2 - 观望 → 等待
1 - 持有 → 继续

卖出信号（从强到弱）:
-1 - 警示 → 观察
-2 - 减仓 → 减半
-3 - 清仓 → 全清
-4 - 强制卖出 → 全清（紧急）
```

### 核心要点

1. **基本面 > 技术面**: 先选好公司，再看买入时机
2. **卖出优先**: 风险控制比收益更重要
3. **仓位管理**: 根据信号级别调整仓位
4. **分散投资**: 不要把鸡蛋放在一个篮子里
5. **持续学习**: 信号系统是辅助工具，不是万能钥匙
