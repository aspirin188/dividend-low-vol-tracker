# 保守策略使用指南

## 问题分析

### 为什么"保守策略"没有数据？

保守策略设置了**非常严格**的筛选条件：

```python
MIN_DIVIDEND_YIELD = 4.0    # 股息率必须 ≥ 4%（非常高）
MIN_MARKET_CAP = 800        # 市值必须 ≥ 800亿（超大市值）
MIN_ROE = 10.0              # ROE必须 ≥ 10%（优秀水平）
MIN_DIVIDEND_YEARS = 5      # 必须连续分红 ≥ 5年（长期稳定）
MAX_DEBT_RATIO = 60         # 负债率必须 ≤ 60%（低风险）
```

### 三重筛选逻辑

系统采用**逐步筛选**机制：

```
原始股票池（348只）
    ↓ 筛选1：股息率 ≥ 4%
剩余股票（约80只）
    ↓ 筛选2：市值 ≥ 800亿
剩余股票（约40只）
    ↓ 筛选3：ROE ≥ 10%
剩余股票（约20只）
    ↓ 筛选4：连续分红 ≥ 5年
剩余股票（约10只）
    ↓ 筛选5：负债率 ≤ 60%
最终结果：可能为 0 只
```

## 使用建议

### ✅ 保守策略适用场景

1. **市场高位**：牛市后期，寻求安全边际
2. **风险厌恶**：极度保守的投资者
3. **资产配置**：作为防守型资产配置

### ⚠️ 保守策略局限性

1. **样本太少**：可能筛选不出任何股票
2. **机会成本**：错过很多优质标的
3. **条件过严**：实际操作中难以执行

### 💡 改进建议

#### 方案1：适度放宽条件（推荐）
```python

MIN_MARKET_CAP = 600       # 市值 ≥ 600亿（大市值）
MIN_ROE = 9.0              # ROE ≥ 9%（优秀）
MIN_DIVIDEND_YEARS = 4     # 连续分红 ≥ 4年
MAX_DEBT_RATIO = 65        # 负债率 ≤ 65%
```

**预期结果**：10-20只高质量股票

#### 方案2：使用均衡策略

#### 方案3：激进策略（追求收益）

## 最佳实践

### ✅ 推荐流程

1. **首次运行**：使用"均衡型"策略（稳妥）
2. **观察结果**：查看筛选出的股票数量
3. **调整策略**：
   - 如果太少（<10只）→ 放宽条件
   - 如果太多（>50只）→ 收紧条件
   - 如果适中（10-50只）→ 深入研究

### ⚠️ 避免误区

❌ **误区1**：盲目追求最严格条件
- 结果：可能筛选不出任何股票
- 失去投资机会

❌ **误区2**：频繁切换策略
- 结果：无法形成稳定的投资体系
- 建议：选定策略后坚持使用

❌ **误区3**：忽略市场环境
- 牛市：可以使用保守策略
- 熊市：应该适度放宽条件

## 附录：策略参数详解

### 股息率下限（MIN_DIVIDEND_YIELD）

- **激进型**：2.0%（覆盖更多股票，但可能质量不高）
- **均衡型**：3.0%（符合红利策略基本要求）
- **保守型**：4.0%（严格筛选，但样本太少）

### 市值下限（MIN_MARKET_CAP）

- **激进型**：300亿（中等市值，成长性好）
- **均衡型**：500亿（大市值，稳定性好）
- **保守型**：800亿（超大市值，但数量少）

### ROE下限（MIN_ROE）

- **激进型**：6.0%（及格线）
- **均衡型**：8.0%（良好）
- **保守型**：10.0%（优秀，但稀缺）

### 分红年数（MIN_DIVIDEND_YEARS）

- **激进型**：2年（短期稳定）
- **均衡型**：3年（中期稳定）
- **保守型**：5年（长期稳定，但很少）

### 负债率上限（MAX_DEBT_RATIO）

- **激进型**：80%（较宽松）
- **均衡型**：70%（适中）
- **保守型**：60%（严格，金融地产除外）
