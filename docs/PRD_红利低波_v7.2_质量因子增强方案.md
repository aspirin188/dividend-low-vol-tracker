## 🔍 新增筛选条件详解

### 1️⃣ 近3年净利润增速 > 0

**问题背景**：
- 周期股可能在盈利高点大额分红，但次年利润下滑
- 一次性收益（资产处置、政府补贴）美化报表
- 分红来自历史利润，不可持续

**筛选逻辑**：
```python
净利润增速计算:
方式1: CAGR（复合增长率）
  (净利润2025 / 净利润2022) ^ (1/3) - 1 > 0

方式2: 逐年对比
  净利润2025 > 净利润2024 > 净利润2023 > 净利润2022

推荐: 方式1（CAGR）
原因: 更科学，容忍单年波动
```

**数据来源**：
- 东方财富API: `get_financial_indicator()`
- 字段: `net_profit`（净利润）
- 时间范围: 近4年年报数据

**配置参数**：
```python
ENABLE_PROFIT_GROWTH_FILTER = True  # 是否启用
MIN_PROFIT_GROWTH_3Y = 0            # 最低增速（0表示正值即可）
```

**案例说明**：
```
❌ 剔除案例:
某钢铁企业
- 2022年净利润: 100亿
- 2023年净利润: 80亿
- 2024年净利润: 60亿
- 2025年净利润: 40亿
- 净利润CAGR: -20.6% → 剔除

✅ 通过案例:
某银行
- 2022年净利润: 800亿
- 2023年净利润: 850亿
- 2024年净利润: 900亿
- 2025年净利润: 950亿
- 净利润CAGR: 5.9% → 通过
```

---

### 2️⃣ 经营现金流/净利润 > 0.8

**问题背景**：
- 分红需要真金白银，不是账面利润
- 部分公司净利润高，但经营现金流为负
- 现金分红靠借款，不可持续

**筛选逻辑**：
```python
现金流质量计算:
经营现金流净额 / 净利润 > 0.8

说明:
- 比值 > 1.0: 盈利质量优秀（真金白银）
- 比值 0.8-1.0: 盈利质量良好（可接受）
- 比值 0.5-0.8: 盈利质量一般（需关注）
- 比值 < 0.5: 盈利质量差（警惕）
- 比值 < 0: 账面盈利，实际亏损（危险）
```

**数据来源**：
- 东方财富API: `get_cashflow_data()`
- 字段: `operating_cashflow`（经营现金流净额）
- 时间范围: 最新年报数据

**配置参数**：
```python
ENABLE_CASHFLOW_QUALITY_FILTER = True  # 是否启用
MIN_CASHFLOW_PROFIT_RATIO = 0.8        # 最低比率
```

**案例说明**：
```
❌ 剔除案例:
某房地产企业
- 净利润: 50亿
- 经营现金流净额: -10亿
- 比值: -0.2 → 剔除

✅ 通过案例:
某高速公路企业
- 净利润: 30亿
- 经营现金流净额: 45亿
- 比值: 1.5 → 通过（盈利质量优秀）
```

---

### 3️⃣ 第一大股东持股比例 > 20%

**问题背景**：
- 股权分散导致决策效率低
- 大股东减持套现风险
- 缺乏利益绑定机制

**筛选逻辑**：
```python
股权结构稳定性:
第一大股东持股比例 > 20%

说明:
- > 50%: 绝对控股（稳定）
- 30%-50%: 相对控股（较稳定）
- 20%-30%: 参股控股（可接受）
- < 20%: 股权分散（不稳定）
```

**数据来源**：
- 东方财富API: `get_shareholder_data()`
- 字段: `top1_shareholder_ratio`（第一大股东持股比例）
- 时间范围: 最新数据

**配置参数**：
```python
ENABLE_SHAREHOLDER_STABILITY_FILTER = True  # 是否启用
MIN_TOP1_SHAREHOLDER_RATIO = 0.2            # 最低持股比例
```

**案例说明**：
```
❌ 剔除案例:
某创业板公司
- 第一大股东持股: 8%
- 前十大股东合计: 25%
- 股权高度分散 → 剔除

✅ 通过案例:
某央企
- 第一大股东持股: 65%
- 实际控制人: 国资委
- 股权结构稳定 → 通过
```

## 🎯 击球区基础版

### 核心功能

**目标**: 判断股票当前价格是否处于合理买入区间

**评分模型**:
```
击球区评分（60分制）:

价格百分位维度（30分）:
├─ 长期百分位 < 20% → 30分
├─ 长期百分位 < 30% → 20分
├─ 长期百分位 < 40% → 10分
└─ 长期百分位 ≥ 40% → 0分

估值维度（30分）:
├─ PE百分位 < 20% → 30分
├─ PE百分位 < 30% → 20分
├─ PE百分位 < 40% → 10分
└─ PE百分位 ≥ 40% → 0分
```

**输出结果**:
```
击球区评级:
├─ 50-60分 → ⭐⭐⭐⭐⭐ 强击球区
├─ 40-50分 → ⭐⭐⭐⭐ 弱击球区
├─ 30-40分 → ⭐⭐⭐ 观察区
├─ 20-30分 → ⭐⭐ 观望区
└─ 0-20分 → ⭐ 高估区
```
