**发布日期**: 2026-03-31

## 📋 版本概述

V8.0是红利低波跟踪系统的一个重要升级版本,在原有的三因子模型基础上,新增质量因子和技术信号,形成五因子智能评分模型。同时引入5种预设策略,大大提升用户体验。

---

## 🎯 主要更新

### 1. 五因子评分模型 ⭐ 核心升级

从三因子模型升级为五因子模型:

| 因子 | 权重 | 新增 | 说明 |
|------|------|------|------|
| 股息率 | 40% | - | TTM股息率 |
| 波动率 | 25% | - | 252日年化波动率 |
| 稳定性 | 15% | - | 分红年数 + 支付率稳定性 |
| **质量因子** ⭐ | **15%** | **新增** | ROE + 利润增长 + 击球区评分 |
| **技术信号** ⭐ | **5%** | **新增** | 10级买卖信号 + 趋势判断 |

### 2. 质量因子模块 ⭐ 新增

**击球区评分系统**:
- ROE评分 (30%)
- 3年利润增长CAGR评分 (20%)
- 资产负债率评分 (20%)
- 股息率评分 (30%)

**评分范围**: 0-100分,分数越高代表股票质量越好

### 3. 技术信号系统 ⭐ 新增

**10级买卖信号**:
- 级别10: 强烈买入 (价格<MA20<MA10<MA5)
- 级别8-9: 买入 (价格>MA5>MA10>MA20)
- 级别6-7: 谨慎买入 (价格>MA10>MA20)
- 级别4-5: 观望 (价格>MA20)
- 级别2-3: 谨慎持有
- 级别0-1: 卖出

**趋势判断**:
- 上升: 20日涨幅>5%
- 下降: 20日涨幅<-5%
- 震荡: -5%≤20日涨幅≤5%

**死叉检测**: 自动识别技术指标死叉信号

### 4. 5种预设策略 ⭐ 新增

用户可以一键应用5种预设策略:

| 策略 | 风险偏好 | 适用人群 |
|------|----------|----------|
| **保守型** | 低风险 | 稳健投资者,追求稳定收益 |
| **均衡型** | 中风险 | 平衡型投资者,兼顾收益与风险 |
| **激进型** | 高风险 | 激进投资者,愿意承担较大波动 |
| **高股息** | 中风险 | 收益型投资者,专注高股息 |
| **低估型** | 中风险 | 价值投资者,寻找低估优质股 |

### 5. 配置历史记录 ⭐ 新增

- 记录所有配置变更
- 显示应用时间和原因
- 支持查看历史配置
- 便于回溯和分析

### 6. 数据源增强 ⭐ 新增

- **价格百分位**: 显示当前价格在历史价格中的位置
- **波动率**: 252日年化波动率计算
- **资产负债率**: 从财务数据中提取
- **分红年数**: 统计连续分红年数

## 💡 使用建议

### 1. 使用预设策略

**首次使用建议**:
- 保守型用户: 选择"保守型"策略
- 平衡型用户: 选择"均衡型"策略
- 了解系统后: 根据需要调整参数

**策略应用步骤**:
1. 访问配置管理页面
2. 选择预设策略
3. 查看策略参数说明
4. 应用策略

### 2. 查看质量因子

**击球区评分解读**:
- 90-100分: 优质股票,值得深入研究
- 80-89分: 良好股票,可以关注
- 70-79分: 一般股票,谨慎选择
- <70分: 需谨慎考虑

### 3. 关注技术信号

**买卖信号解读**:
- 级别8-10: 考虑买入
- 级别4-7: 观望
- 级别0-3: 考虑卖出

**趋势判断**:
- 上升趋势: 适合持有
- 下降趋势: 谨慎持有
- 震荡趋势: 保持观察

### 4. 结合五因子综合判断

不要只看单一因子,建议:
1. 综合评分: 关注排名靠前的股票
2. 质量因子: 优先选择高分股票
3. 技术信号: 关注买入信号
4. 趋势判断: 选择上升趋势

## ❓ 常见问题

### Q1: 五因子模型是否更准确?

A: 五因子模型在原有基础上增加了质量因子和技术信号,能够更全面地评估股票质量和技术面情况,理论上更准确。但实际效果需要市场验证。

### Q2: 如何选择预设策略?

A: 根据您的风险偏好选择:
- 保守型: 低风险,追求稳定收益
- 均衡型: 中风险,兼顾收益与风险
- 激进型: 高风险,追求高收益
- 高股息: 专注高股息率
- 低估型: 寻找低估优质股

### Q3: 质量因子如何计算?

A: 击球区评分综合考虑:
- ROE (30%): 净资产收益率
- 利润增长CAGR (20%): 3年复合增长率
- 资产负债率 (20%): 财务风险
- 股息率 (30%): 分红收益

### Q4: 技术信号可靠吗?

A: 技术信号仅供参考,不能作为唯一买卖依据。建议结合基本面分析和质量因子综合判断。

### Q5: 如何回退到旧版本?

A: 如需回退,可以:
1. 使用git回退代码
2. 恢复数据库备份
3. 重启服务
