## 4. 两因子评分模型

### 4.1 数据口径定义

```
┌──────────────────────────────────────────────────────┐
│              数据口径 — 写死，不可配置                   │
├──────────────────────────────────────────────────────┤
│  股息率：TTM（近12个月）                               │
│  波动率：120个交易日对数收益率，年化 √242               │
│  总市值：最新实时行情（亿元）                           │
│  股利支付率：每股分红 / EPS × 100%                     │
│  股价：push2 f2（最新价）                              │
│  市盈率(PE)：push2 f9（动态）                          │
│  市净率(PB)：push2 f23                                │
└──────────────────────────────────────────────────────┘
```

### 4.2 评分算法

```
综合评分 = 股息率归一化 × 0.6 + (1 - 波动率归一化) × 0.4
归一化方式：min-max，映射到 0-100 分
```

### 4.3 硬性筛选条件

```
1. 股息率(TTM) ≥ 4.0%
2. 总市值 ≥ 100亿
3. 股利支付率 ≤ 150%（且 EPS > 0）
4. 当年 EPS > 0
5. 非ST股票
6. 数据完整性：股息率、波动率、市值、EPS 均非空
```

## 6. 极简技术方案

### 6.1 技术栈

| 层 | 技术选型 |
|----|---------|
| 后端 | Python 3.9+ / Flask |
| 前端 | HTML + CSS + JavaScript（原生） |
| 数据处理 | pandas, numpy |
| 股票数据 | akshare |
| Excel导出 | openpyxl |
| 数据存储 | SQLite |

### 6.2 API 列表（无变化）

| API | 方法 | 说明 |
|-----|------|------|
| `GET /` | GET | 渲染首页 |
| `POST /api/run` | POST | 一键运行（同步，超时10分钟） |
| `GET /api/stocks` | GET | 获取标的列表 |
| `GET /api/export` | GET | 导出Excel |

### 6.3 列排序实现方案（v6.3 新增）

```
方案：前端纯 JavaScript 排序，不请求后端。

实现方式：
1. loadStocks() 获取全量数据后缓存到全局变量 allStocks
2. 筛选/搜索后得到 filteredStocks（用于排序的源数据）
3. 点击表头时，对 filteredStocks 在前端排序并重新渲染
4. 排序状态用全局变量 sortColumn / sortDirection 跟踪

排序逻辑：
- 数值列（price, pe, pb, dividend_yield, annual_vol, composite_score,
  market_cap, payout_ratio, eps, rank）：按数值比较
- 文本列（name, code, industry）：按 localeCompare 比较
- 空值统一排最后
- 三态循环：升序 → 降序 → 默认（综合评分降序）
```

### 6.4 数据表结构（v6.2 定义，无变化）

```sql
CREATE TABLE stock_data (
    code             TEXT PRIMARY KEY,
    name             TEXT,
    industry         TEXT,
    dividend_yield   REAL,
    annual_vol       REAL,
    composite_score  REAL,
    rank             INTEGER,
    market_cap       REAL,
    payout_ratio     REAL,
    eps              REAL,
    price            REAL,
    pe               REAL,
    pb               REAL,
    data_date        TEXT,
    updated_at       TEXT
);
```

### 6.5 Excel 导出格式（13列，无变化）

```
排名 | 代码 | 名称 | 行业 | 股价 | PE | PB | 股息率(%) | 波动率(%) | 综合评分 | 总市值(亿) | 股利支付率(%) | EPS
```
