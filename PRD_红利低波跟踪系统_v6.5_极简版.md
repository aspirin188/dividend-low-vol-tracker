## 3. 两因子评分模型

```
综合评分 = 股息率归一化 × 0.6 + (1 - 波动率归一化) × 0.4

数据口径：
- 股息率：TTM（近12个月）
- 波动率：120日对数收益率，年化 √242

硬性筛选：
- 股息率 ≥ 4%
- 总市值 ≥ 100亿
- 股利支付率 ≤ 150%
- EPS > 0
- 非ST
```

## 5. 技术方案

### 5.1 数据表结构（16 字段）

```sql
CREATE TABLE stock_data (
    code             TEXT PRIMARY KEY,
    name             TEXT,
    industry         TEXT,       -- 归并后的大类行业（v6.5 变更）
    market           TEXT,       -- 市场类型：沪市/深市/创业板/科创板（v6.5 新增）
    dividend_yield   REAL,
    annual_vol       REAL,
    composite_score  REAL,
    rank             INTEGER,
    market_cap       REAL,
    payout_ratio     REAL,
    eps              REAL,
    price            REAL,
    pe               REAL,
    pb               REAL,
    data_date        TEXT,
    updated_at       TEXT
);
```

### 5.2 API 筛选参数

```
/api/stocks 支持 query 参数：

  market_cap_range  市值区间
    "large"         ≥ 1000 亿
    "medium"        500 - 1000 亿
    "small"         100 - 500 亿

  div_yield_min     股息率下限
    "4" / "6" / "8"

  market            市场类型（v6.5 新增）
    "sh"            沪市主板
    "sz"            深市主板
    "gem"           创业板
    "star"          科创板
```

## 6. 验收标准

| 验收项 | 标准 |
|--------|------|
| 行业显示 | 显示大类行业名称（如"化工"而非"化学制品"），筛选器选项为大类 |

| 数据一致性 | 行业筛选、列表显示、Excel导出三处行业名称一致 |
