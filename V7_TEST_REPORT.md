## 四、新功能验证

### 4.1 支付率稳定性评分

**实现方式**:
```python
stability_score = 分红年数分 × 0.6 + 支付率稳定性分 × 0.4
```

**评分分级**:
- 30%-80%: 100分（理想区间）
- 80%-100%: 80分（合理区间）
- 100%-150%: 50分（警告区间）
- >150%: 0分（危险区间）

**实际案例**:
- 中远海控: 支付率3年均值50.28% → 评分100分
- 兴业银行: 支付率3年均值30.5% → 评分100分
- 双汇发展: 支付率3年均值98.17% → 评分80分

### 4.2 波动率窗口配置化

**配置项**:
- 参数名: `VOL_WINDOW`
- 默认值: 120日
- 范围: 120-252日
- 单位: 交易日

**效果**:
- 120日: 响应快，适合短期策略
- 252日: 更稳定，适合长期策略
