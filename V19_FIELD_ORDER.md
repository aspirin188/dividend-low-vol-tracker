# v6.19 字段顺序调整说明

## 📅 调整日期
2026-03-29 16:10

---

## 🎯 调整目标

将**股价百分位**字段从最后一列移到**综合评分**后面，使字段顺序更符合逻辑。

---

## 📊 调整前后对比

### 调整前

```
排名 → 名称 → 代码 → 行业 → 市场 → 股价 → PE → PB → 股息率 → 市值 → 综合评分 → 波动率 → 支付率 → EPS → 分红年数 → ROE → 负债率 → 股价百分位
                                                                                                                                                    ↑ 最后
```

### 调整后

```
排名 → 名称 → 代码 → 行业 → 市场 → 股价 → PE → PB → 股息率 → 市值 → 综合评分 → 股价百分位 → 波动率 → 支付率 → EPS → 分红年数 → ROE → 负债率
                                                                                     ↑ 移到这里
```

---

## 💡 调整原因

### 1. 逻辑关联性

**股价百分位**与**综合评分**紧密相关：
- 综合评分：反映股票整体质量
- 股价百分位：反映当前价格位置
- 两者结合：判断是否值得买入

**示例**：
```
综合评分: 85分 (高质量)
股价百分位: 20% (相对低位)
→ 结论: 好公司，好价格，值得关注
```

### 2. 用户体验

- **核心字段前置**：综合评分和股价百分位是决策的关键指标
- **减少横向滚动**：用户无需滚动到最后才能看到股价百分位
- **符合查看习惯**：先看评分，再看价格位置

### 3. 功能分组

调整后的字段分组更清晰：

| 分组 | 字段 | 说明 |
|------|------|------|
| **基础信息** | 排名、名称、代码、行业、市场 | 股票基本信息 |
| **估值指标** | 股价、PE、PB | 估值判断 |
| **核心指标** | 股息率、市值、综合评分、股价百分位 | 决策关键 |
| **财务指标** | 波动率、支付率、EPS、分红年数、ROE、负债率 | 辅助分析 |
