# 红利低波跟踪系统 v7.2 (Dividend Low-Volatility Tracker)

基于三因子模型的高股息股票筛选系统

## 📊 项目简介

红利低波跟踪系统是一个自动化股票筛选工具，通过**股息率+低波动率+分红稳定性**三因子模型，帮助投资者发现高质量的高股息投资标的。

### 核心功能

- ✅ **一键运行**: 自动获取数据、筛选、评分
- ✅ **多维筛选**: 支持行业、市值、股息率、市场、ROE、负债率、支付率、股价百分位等筛选
- ✅ **数据验证**: 双重数据源验证，确保数据准确性
- ✅ **健康度追踪**: 实时监控数据源可靠性，透明展示数据质量
- ✅ **Excel导出**: 一键导出筛选结果
- ✅ **综合评分**: 三因子模型自动评分排名

| **配置化参数** | 灵活调整，适应不同风格 |

### 三因子评分模型

**评分公式**：
```
综合评分 = 股息率因子 × 50% + 波动率因子 × 30% + 稳定性因子 × 20%
```

**因子说明**：

| 因子 | 权重 | 数据口径 | 评分逻辑 |
|------|------|----------|----------|
| **股息率** | 50% | TTM（近12个月自计算） | Min-Max归一化，越高越好 |
| **波动率** | 30% | 120日对数收益率年化 | 反向评分，越低越好 |
| **稳定性** | 20% | 分红年数 + 支付率稳定性 | 综合评分（v7.0升级） |

**稳定性因子升级（v7.0）**：
```
稳定性评分 = 分红年数评分 × 60% + 支付率稳定性评分 × 40%
```

- **分红年数**：连续5年=100分，4年=80分，3年=60分
- **支付率稳定性**：3年平均支付率
  - 30%-80% → 100分（理想区间）
  - 80%-100% → 80分（合理区间）
  - 100%-150% → 50分（警告区间）
  - >150% → 0分（危险区间）

### 硬性筛选条件

**"宁可错过，不可做错"** - 评分前先过滤风险

| 条件 | 标准 | 说明 |
|------|------|------|
| 股息率 | 3%-10% | 确保回报，避免陷阱 |
| 总市值 | ≥500亿 | 确保流动性 |
| 股利支付率 | ≤150% | 避免不可持续分红 |
| EPS | >0 | 排除亏损公司 |
| ST标识 | 无 | 排除风险警示股 |
| 连续分红 | ≥3年 | 验证分红意愿 |
| ROE | ≥8% | 筛选盈利能力 |
| 资产负债率 | ≤70% | 控制财务风险 |
| 资产负债率（金融地产） | ≤85% | 行业特殊处理 |
| 近3年净利润增速 ⭐ v7.2 | >0 | 防止周期股美化报表 |
| 经营现金流/净利润 ⭐ v7.2 | >0.8 | 防止现金分红靠借款 |
| 第一大股东持股比例 ⭐ v7.2 | >20% | 确保股权结构稳定 |

### v7.0核心亮点

**1. 支付率稳定性维度** ⭐⭐⭐⭐⭐
- **问题**：单年支付率易被操纵，支付率>100%不可持续
- **解决**：计算3年平均支付率，分4档评分
- **价值**：验证"真金白银"分红，识别一次性特别分红

**2. 波动率窗口配置化** ⭐⭐⭐⭐
- **问题**：固定窗口不适配所有投资风格
- **解决**：120日（波段操作）/252日（长期持有）可选
- **价值**：灵活适应不同投资策略

**3. 股息率异常检测** ⭐⭐⭐
- **问题**：TTM股息率容易被一次性分红误导
- **解决**：检测TTM/3年均值比值，>3.0降权处理
- **价值**：避免踩雷特别分红陷阱

### v7.2核心亮点

**1. 净利润增速筛选** ⭐⭐⭐⭐⭐
- **问题**：周期股在盈利高点大额分红，次年利润下滑
- **解决**：近3年净利润CAGR > 0，验证盈利持续性
- **价值**：防止周期股美化报表，避免分红来自历史利润

**2. 现金流质量筛选** ⭐⭐⭐⭐⭐
- **问题**：净利润高但经营现金流为负，现金分红靠借款
- **解决**：经营现金流/净利润 > 0.8，验证分红可持续性
- **价值**：确保分红来自真金白银，而非账面盈利

**3. 股权结构筛选** ⭐⭐⭐⭐
- **问题**：股权分散导致决策效率低，大股东减持风险
- **解决**：第一大股东持股 > 20%，确保股权结构稳定
- **价值**：大股东与中小股东利益绑定，降低恶意减持风险

**4. 击球区评分** ⭐⭐⭐⭐
- **问题**：选股后缺乏择时参考，不知何时买入
- **解决**：价格百分位+估值百分位综合评分（60分制）
- **价值**：辅助买入时机决策，识别"击球区"

### 选股流程

```
5000+ A股
    ↓ 硬性筛选（11个条件）
    ↓ 三因子评分
    ↓ 击球区评分（v7.2新增）
    ↓ 综合排名
12-15只高质量股票
```

**典型筛选效果**：
- 从5518只股票中筛选出33只
- 覆盖金融、能源、制造等稳健行业
- 平均股息率5%+，平均波动率20%-

### 关键指标计算

#### 1. 股息率（TTM）

```python
TTM股息率 = 过去12个月每股股利总和 / 当前股价 × 100%

数据源: 东方财富分红配送数据（stock_fhps_em）
计算方式: 自计算，不依赖第三方字段
```

#### 2. 波动率

```python
年化波动率 = std(对数收益率) × √242

窗口期: 120个交易日
数据源: akshare股票历史数据
```

#### 3. 分红稳定性

```python
连续分红年数 = 从最近一年往前，连续有分红的年份数

判断标准: 包含"派"（现金分红）或"送"（股票分红）
数据源: 东方财富分红配送数据
```

#### 4. ROE（净资产收益率）

```python
ROE = 净利润 / 净资产 × 100%

数据源: akshare业绩报表（stock_yjbb_em）
完整性: 99.4%
```

#### 5. 资产负债率

```python
资产负债率 = 总负债 / 总资产 × 100%

计算方式: 从资产负债表获取数据计算
支持: 银行股和普通股票
```

#### 6. 股价百分位

```python
股价百分位 = 当前价格在历史250日价格序列中的百分位

含义:
- 0-20: 低位区（股价低于历史80%的时间）
- 20-40: 中低位
- 40-60: 中位区
- 60-80: 中高位
- 80-100: 高位区
```
