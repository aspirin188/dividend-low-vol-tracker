# 红利低波跟踪系统 (Dividend Low-Volatility Tracker)

基于三因子模型的高股息股票筛选系统

## 📊 项目简介

红利低波跟踪系统是一个自动化股票筛选工具，通过**股息率+低波动率+分红稳定性**三因子模型，帮助投资者发现高质量的高股息投资标的。

### 核心功能

- ✅ **一键运行**: 自动获取数据、筛选、评分
- ✅ **多维筛选**: 支持行业、市值、股息率、市场、ROE、负债率、支付率、股价百分位等筛选
- ✅ **数据验证**: 双重数据源验证，确保数据准确性
- ✅ **健康度追踪**: 实时监控数据源可靠性，透明展示数据质量
- ✅ **Excel导出**: 一键导出筛选结果
- ✅ **综合评分**: 三因子模型自动评分排名

### 三因子模型

```
综合评分 = 股息率归一化 × 50% + (1 - 波动率归一化) × 30% + 分红稳定性归一化 × 20%
```

**数据口径:**
- **股息率**: TTM（近12个月，自计算）
- **波动率**: 120日对数收益率，年化 √242
- **分红稳定性**: 过去5年连续分红年数

**硬性筛选条件 (v6.19):**
- 股息率 ≥ 3%
- 总市值 ≥ 500亿
- 股利支付率 ≤ 150%
- EPS > 0
- 非ST股票
- 连续分红年数 ≥ 3年
- ROE ≥ 8%
- 资产负债率 ≤ 70%（金融地产 ≤ 85%）

### 重启系统

如需重新初始化系统（清除旧数据，重新获取）：

#### 方法1: 完整重启（推荐）


**注意**: 方法2不会清除数据库，只是重新获取数据。如果需要完全重置，请使用方法1。

| **分红年数** | 东方财富分红配送 | - | ⭐⭐⭐⭐ |
| **股价百分位** | akshare历史数据 | - | ⭐⭐⭐⭐ |

### 关键指标计算

#### 1. 股息率（TTM）

```python
TTM股息率 = 过去12个月每股股利总和 / 当前股价 × 100%

数据源: 东方财富分红配送数据（stock_fhps_em）
计算方式: 自计算，不依赖第三方字段
```

#### 2. 波动率

```python
年化波动率 = std(对数收益率) × √242

窗口期: 120个交易日
数据源: akshare股票历史数据
```

#### 3. 分红稳定性

```python
连续分红年数 = 从最近一年往前，连续有分红的年份数

判断标准: 包含"派"（现金分红）或"送"（股票分红）
数据源: 东方财富分红配送数据
```

#### 4. ROE（净资产收益率）

```python
ROE = 净利润 / 净资产 × 100%

数据源: akshare业绩报表（stock_yjbb_em）
完整性: 99.4%
```

#### 5. 资产负债率

```python
资产负债率 = 总负债 / 总资产 × 100%

计算方式: 从资产负债表获取数据计算
支持: 银行股和普通股票
```

#### 6. 股价百分位

```python
股价百分位 = 当前价格在历史250日价格序列中的百分位

含义:
- 0-20: 低位区（股价低于历史80%的时间）
- 20-40: 中低位
- 40-60: 中位区
- 60-80: 中高位
- 80-100: 高位区
```
