### 三、实施细节

#### 1. ROE数据源实现

**数据源A（主）：akshare yjbb**
```python
def fetch_roe_from_yjbb(code, context):
    """从akshare yjbb获取ROE"""
    eps_df = context.get('eps_df')
    if eps_df is not None and code in eps_df['code'].values:
        return eps_df[eps_df['code'] == code]['roe'].values[0]
    return None
```

**数据源B（副）：计算方式**
```python
def fetch_roe_by_calculation(code, context):
    """通过计算获取ROE"""
    # 1. 获取利润表
    profit_df = ak.stock_financial_report_sina(stock=code, symbol="利润表")
    
    # 2. 提取净利润（需要匹配多种字段名）
    net_profit = extract_value(profit_df, [
        '净利润',
        '归母净利润',
        '归属于母公司股东的净利润'
    ])
    
    # 3. 获取资产负债表
    balance_df = ak.stock_financial_report_sina(stock=code, symbol="资产负债表")
    
    # 4. 提取股东权益（需要匹配多种字段名）
    net_assets = extract_value(balance_df, [
        '股东权益合计',
        '所有者权益合计',
        '归属于母公司股东权益合计'
    ])
    
    # 5. 计算ROE
    if net_profit and net_assets and net_assets > 0:
        roe = (net_profit / net_assets) * 100
        return round(roe, 2)
    
    return None
```

#### 2. 改进点

**问题1：银行股股东权益提取失败**

原因：银行股的资产负债表格式不同，字段名可能为：
- "股东权益合计"
- "所有者权益合计"
- "归属于母公司股东权益合计"
- "股东权益"

解决方案：
```python
def extract_net_assets(balance_df):
    """提取股东权益（支持多种字段名）"""
    keywords = [
        '股东权益合计',
        '所有者权益合计',
        '归属于母公司股东权益合计',
        '股东权益',
        '净资产',
        '所有者权益'
    ]
    
    for keyword in keywords:
        for idx, row in balance_df.iterrows():
            if keyword in str(row.iloc[0]):
                return parse_value(row.iloc[1])
    
    return None
```

**问题2：ROE计算差异大**

原因：
- akshare yjbb可能使用"加权ROE"
- 计算方式可能使用"稀释ROE"
- 净利润可能用"归母净利润" vs "净利润"
- 股东权益可能用"归母股东权益" vs "股东权益合计"

解决方案：
1. 统一计算公式，确保与akshare yjbb一致
2. 或者接受一定差异，设置合理的容忍度
3. 对差异大的数据进行标记和人工审核
