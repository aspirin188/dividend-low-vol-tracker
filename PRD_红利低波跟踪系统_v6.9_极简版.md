### 功能详情

#### 1. "复原"按钮功能

**位置**: 筛选器行末尾

**功能行为**:
- 清空搜索框内容
- 重置所有筛选器（行业、市值、股息率、市场）为默认值
- 重置表格排序为默认状态（综合评分降序）
- 自动刷新表格数据

**交互设计**:
```
┌──────────────────────────────────────────────────────────────────┐
│  [搜索名称/代码...]                          [ 复原 ]          │
│  [全部行业 ▾] [全部市值 ▾] [全部股息率 ▾] [全部市场 ▾]          │
└──────────────────────────────────────────────────────────────────┘
```

#### 2. 简拼搜索功能

**功能定义**:
- 支持输入股票名称的首个拼音字母进行搜索
- 例如：输入"zgyh"可匹配"中国银行"，输入"jsyh"可匹配"建设银行"

**实现方案**:
- 后端提供股票拼音首字母字段（pinyin_abbr）
- 前端搜索时同时匹配：名称、代码、拼音首字母
- API搜索参数扩展：`q` 参数同时匹配名称、代码、拼音

**数据结构扩展**:
```sql
-- 数据库新增字段
ALTER TABLE stock_data ADD COLUMN pinyin_abbr TEXT;  -- 拼音首字母缩写
```

**API返回扩展**:
```json
{
  "code": "601939",
  "name": "建设银行",
  "pinyin_abbr": "jsyh",  // 新增
  ...
}
```

## 3. 两因子评分模型

```
综合评分 = 股息率归一化 × 0.6 + (1 - 波动率归一化) × 0.4

数据口径:
- 股息率:TTM(近12个月,自计算)
- 波动率:120日对数收益率,年化 √242

硬性筛选:
- 股息率 ≥ 3%
- 总市值 ≥ 500亿
- 股利支付率 ≤ 150%
- EPS > 0
- 非ST
```

## 5. 技术方案

### 5.1 数据表结构(v6.9 新增pinyin_abbr字段)

```sql
CREATE TABLE stock_data (
    code             TEXT PRIMARY KEY,
    name             TEXT,
    industry         TEXT,       -- 归并后的大类行业
    market           TEXT,       -- 市场类型:沪市/深市/创业板/科创板
    dividend_yield   REAL,       -- 自计算TTM股息率
    annual_vol       REAL,
    composite_score  REAL,
    rank             INTEGER,
    market_cap       REAL,
    payout_ratio     REAL,
    eps              REAL,
    price            REAL,
    pe               REAL,
    pb               REAL,
    pinyin_abbr      TEXT,       -- v6.9新增: 拼音首字母缩写
    data_date        TEXT,
    updated_at       TEXT
);
```

### 5.2 筛选条件配置

```python
# server/services/scorer.py

MIN_DIVIDEND_YIELD = 3.0   # 股息率 ≥ 3%
MIN_MARKET_CAP = 500.0      # 总市值 ≥ 500亿
MAX_PAYOUT_RATIO = 150.0    # 股利支付率 ≤ 150%
MIN_EPS = 0.0               # EPS > 0
```

### 5.3 拼音转换方案

使用 Python `pypinyin` 库：
```python
from pypinyin import lazy_pinyin

def get_pinyin_abbr(name):
    """获取股票名称的拼音首字母"""
    return ''.join([p[0] for p in lazy_pinyin(name)])
```
